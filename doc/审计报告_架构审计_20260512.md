### 2.1 ❌ 问题1：两套数据源架构并存

**当前状态**：
- `data_service/collectors/` - 旧架构，包含大量重复代码

## 三、建议方案

### 方案A：渐进式重构（推荐）

**阶段1**：清理混乱
```
1. 删除或归档旧的 collectors/ 下的重复适配器
2. 统一测试文件到 tests/ 目录
3. 更新文档
```

**阶段2**：统一新架构
```
sources/
  ├── qq_realtime.py         (新)
  ├── telegram_realtime.py  (新)
  ├── twitter_realtime.py   (需改造)
  └── ...

examples/
  ├── qq_mvp.py
  └── telegram_mvp.py
```

**阶段3**：集成 Event Bus
```
QQ/Telegram 实时源
      ↓
  StandardEvent
      ↓
  Event Bus
      ↓
  其他服务订阅
```
