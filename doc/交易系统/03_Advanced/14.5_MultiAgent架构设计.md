# TradeAgent 多Agent架构设计文档

---

# 🧠 一、设计背景

BettaFish（微舆）是一个典型的多 Agent 协作系统，通过多个专用 Agent 并行工作 + ForumEngine 协调层做决策聚合。

TradeAgent 借鉴这个模式，将系统中的数据采集、特征计算、情绪分析等功能 Agent 化，提升并行效率和容错能力。

# 🤖 三、Agent 接口规范

## 3.1 Agent 类型总览

| Agent 类型 | 输入 | 输出 | 功能描述 |
|-----------|------|------|----------|
| **Data Agent** | 数据源配置 | 标准化 JSON | 并行抓取行情/ETF/宏观/新闻 |
| **Feature Agent** | 原始 JSON | 数值特征 | 计算价格/资金/动量/波动率 |
| **Sentiment Agent** | 新闻/社交文本 | 结构化情绪 | 分析情绪/识别事件类型 |
| **Explain Agent** | 因子/信号 | 自然语言 | 输出交易信号原因 |
| **Regime Agent** | 价格/因子/风险 | 市场状态 | 判断市场宏观环境 |
| **Control Agent** | Telegram指令 | 执行反馈 | 处理外部控制指令 |

## 3.2 Data Agent（数据代理）

```python
class DataAgent(Agent):
    """数据采集Agent基类"""

    def __init__(self, name, source_config):
        self.name = name
        self.source = source_config

    async def fetch(self):
        raise NotImplementedError

    async def parse(self, raw_data):
        raise NotImplementedError

    async def run(self):
        raw = await self.fetch()
        return await self.parse(raw)

    def get_capabilities(self):
        return {
            "type": "data",
            "name": self.name,
            "sources": list(self.source.keys())
        }
```

## 3.4 Sentiment Agent（情绪代理）

```python
class SentimentAgent(Agent):
    """情绪分析Agent - 使用LLM"""

    def __init__(self, llm_service):
        self.llm = llm_service

    async def analyze(self, news_items):
        results = []

        for news in news_items:
            sentiment = await self.llm.analyze(news["text"])
            results.append({
                "news": news,
                "sentiment": sentiment["score"],
                "event_type": sentiment["event_type"],
                "impact": sentiment["impact"]
            })

        return self.aggregate(results)

    def aggregate(self, results):
        total = sum(r["sentiment"] * r["impact"] for r in results)
        weight = sum(r["impact"] for r in results)

        return {
            "type": "sentiment",
            "sentiment": total / weight if weight > 0 else 0,
            "event_types": list(set(r["event_type"] for r in results)),
            "news_count": len(results),
            "confidence": min(1.0, len(results) / 10)
        }

    def get_capabilities(self):
        return {
            "type": "sentiment",
            "name": "SentimentAgent",
            "llm_model": self.llm.model_name
        }
```

  "sentiment": -0.6,

  "confidence": 0.85
}

## 3.5 Regime Agent（状态代理）

```python
class RegimeAgent(Agent):
    """市场状态判断Agent"""

    def __init__(self, regime_rules):
        self.rules = regime_rules

    async def classify(self, features, risk_index):
        scores = self.compute_scores(features, risk_index)
        regime = max(scores.items(), key=lambda x: x[1])[0]

        return {
            "type": "regime",
            "regime": regime,
            "scores": scores,
            "confidence": self.compute_confidence(scores),
            "rules_applied": self.rules[regime]
        }

    def get_capabilities(self):
        return {
            "type": "regime",
            "name": "RegimeAgent",
            "supported_regimes": list(self.rules.keys())
        }
```

## 3.7 Control Agent（控制代理）

```python
class ControlAgent(Agent):
    """控制指令Agent"""

    def __init__(self, command_handlers):
        self.handlers = command_handlers

    async def receive(self, message):
        cmd = self.parse_command(message)

        handler = self.handlers.get(cmd["action"])
        if handler:
            result = await handler.execute(cmd["params"])
            return {"status": "success", "result": result}
        else:
            return {"status": "error", "message": "unknown_command"}

    def parse_command(self, message):
        return {
            "action": message.payload["action"],
            "params": message.payload.get("params", {}),
            "user": message.payload["user"]
        }

    def get_capabilities(self):
        return {
            "type": "control",
            "name": "ControlAgent",
            "supported_actions": list(self.handlers.keys())
        }
```

# 🔄 四、Agent 协调层设计

## 4.1 Orchestrator（编排器）

```python
class AgentOrchestrator:
    """多Agent协调器"""

    def __init__(self):
        self.data_agents = []
        self.feature_agents = []
        self.sentiment_agent = None
        self.regime_agent = None
        self.explain_agent = None
        self.message_bus = MessageBus()
        self.task_queue = TaskQueue()

    async def initialize(self):
        """初始化所有Agent"""
        self.data_agents = [
            PriceAgent("BTC", {...}),
            PriceAgent("ETH", {...}),
            ETFAgent({...}),
            MacroAgent({...}),
            NewsAgent({...}),
        ]

        self.feature_agents = [
            TrendAgent("trend"),
            VolAgent("volatility"),
            FlowAgent("flow"),
            BehaviorAgent("behavioral"),
        ]

        self.sentiment_agent = SentimentAgent(llm_service)
        self.regime_agent = RegimeAgent(REGIME_RULES)
        self.explain_agent = ExplainAgent(llm_service)

    async def run_full_cycle(self):
        """完整运行周期"""
        start = time.time()

        # 1. 并行采集数据
        data_results = await self.gather_data()

        # 2. 并行计算特征
        feature_results = await self.compute_features(data_results)

        # 3. 情绪分析
        sentiment = await self.sentiment_agent.run(data_results["news"])

        # 4. 状态判断
        regime = await self.regime_agent.classify(feature_results, sentiment)

        # 5. 决策（简单规则）
        decision = self.make_decision(feature_results, sentiment, regime)

        # 6. 解释
        explanation = await self.explain_agent.explain(
            decision, feature_results, regime
        )

        elapsed = time.time() - start

        return {
            "decision": decision,
            "explanation": explanation,
            "regime": regime,
            "factors": feature_results,
            "sentiment": sentiment,
            "cycle_time_ms": elapsed * 1000
        }

    async def gather_data(self):
        """并行采集数据"""
        tasks = [agent.run() for agent in self.data_agents]
        results = await asyncio.gather(*tasks, return_exceptions=True)

        return {agent.name: r for agent, r in zip(self.data_agents, results)}

    async def compute_features(self, data):
        """并行计算特征"""
        tasks = [agent.compute(data) for agent in self.feature_agents]
        results = await asyncio.gather(*tasks, return_exceptions=True)

        return {agent.factor_name: r for agent, r in zip(self.feature_agents, results)}
```

## 4.2 任务调度策略

```python
class TaskScheduler:
    """任务调度器"""

    PRIORITIES = {
        "price": 1,      # 最高优先级
        "etf": 2,
        "macro": 3,
        "news": 4,       # 最低优先级
    }

    def schedule(self, tasks):
        """按优先级排序任务"""
        return sorted(tasks, key=lambda t: self.PRIORITIES.get(t.type, 99))

    def parallelize(self, tasks, max_parallel=5):
        """分批并行执行"""
        batches = []
        for i in range(0, len(tasks), max_parallel):
            batches.append(tasks[i:i+max_parallel])
        return batches
```

## 4.3 消息总线

```python
class MessageBus:
    """Agent消息总线"""

    def __init__(self):
        self.subscribers = defaultdict(list)
        self.queue = asyncio.Queue()

    def subscribe(self, agent, msg_types):
        for msg_type in msg_types:
            self.subscribers[msg_type].append(agent)

    async def publish(self, msg):
        await self.queue.put(msg)

    async def dispatch(self):
        """分发消息到订阅者"""
        while True:
            msg = await self.queue.get()

            for agent in self.subscribers.get(msg.type, []):
                try:
                    await agent.receive(msg)
                except Exception as e:
                    logger.error(f"Agent {agent.name} failed to receive: {e}")
```

## 4.4 生命周期管理

```python
class AgentLifecycle:
    """Agent生命周期管理"""

    def __init__(self):
        self.agents = {}
        self.restart_policy = {
            "max_retries": 3,
            "backoff_base": 2,
        }

    async def start(self, agent):
        """启动Agent"""
        self.agents[agent.name] = {
            "agent": agent,
            "status": "running",
            "restarts": 0,
            "last_error": None
        }
        asyncio.create_task(agent.run())

    async def stop(self, agent_name):
        """停止Agent"""
        if agent_name in self.agents:
            self.agents[agent_name]["status"] = "stopped"

    async def restart(self, agent_name):
        """重启Agent"""
        agent_info = self.agents.get(agent_name)
        if not agent_info:
            return

        agent_info["restarts"] += 1

        if agent_info["restarts"] < self.restart_policy["max_retries"]:
            await self.stop(agent_name)
            await asyncio.sleep(self.restart_policy["backoff_base"] ** agent_info["restarts"])
            await self.start(agent_info["agent"])
```

# ⚠️ 七、设计边界

## 7.1 必须遵守的原则

```
❌ 不要让 LLM Agent 直接控制交易逻辑
   → 主力是规则 + 风险模型

❌ Agent 输出不能直接下单
   → 必须经过 Position Engine 验证

❌ Agent 不能绕过风控
   → 所有交易必须通过 Risk Engine

❌ Agent 之间不能循环依赖
   → 必须保持单向数据流
```

## 7.2 正确用法

```
✅ Data Agent → 并行采集多源数据
✅ Feature Agent → 计算数值特征
✅ Sentiment Agent → LLM 文本理解（输出结构化）
✅ Explain Agent → 生成自然语言解释
✅ Control Agent → Telegram/OpenClaw 指令解析
```
