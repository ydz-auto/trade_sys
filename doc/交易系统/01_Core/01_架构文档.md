# 🧠 一句话理解 TradeAgent

```
TradeAgent = 一个"自动做交易决策并执行"的系统
（输入市场 → 输出交易 → 自我调整）
```

# 🔄 三、数据流（系统"血液循环"）

```
   ↓

│  Risk反馈     │

│  Regime反馈   │
└───────┬───────┘
        ↓
    调整 Risk / Decision（闭环）
```

# 🔄 四、反馈闭环（Feedback Loop）

## 4.1 四大反馈机制

```
┌─────────────────────────────────────────────────────────┐
│                   Feedback Loop                          │
│                                                         │
│  1. PnL Feedback（盈亏反馈）                            │
│     → 连续亏损 → 降低仓位                               │
│                                                         │
│  2. Risk Feedback（风险反馈）                           │
│     → 高风险低回报 → 收紧风险                           │
│                                                         │
│  3. Signal Feedback（信号反馈）                         │
│     → B级信号不准 → 提高执行门槛                        │
│                                                         │
│  4. Regime Feedback（状态反馈）                         │
│     → Regime预测不准 → 调整权重                         │
└─────────────────────────────────────────────────────────┘
```

## 4.2 反馈规则

| 触发条件 | 响应动作 |
|----------|----------|
| 连续亏损3次 | 降低仓位50% |
| 连续亏损5次 | 降低仓位70% |
| 单日亏损5% | 停止交易 |
| 回撤10% | 紧急止损 |
| B级信号准确率<40% | 提高质量阈值 |

```
现在是：
趋势？震荡？恐慌？

组合暴露是否过高？
```

```

```

## 5️⃣ 执行层（Execution）

```
下单
止损止盈
跟踪订单
回测/实盘隔离

对接：Binance / OKX
```

## 6️⃣ 状态 + 反馈（State + Feedback）

```
当前仓位
盈亏状态
风险状态
策略版本
```

# 🧵 六、三条主线

## 主线1：决策主线（赚钱核心）

```
Feature → Factor → Regime → Risk → Decision
```

## 主线2：执行主线（把钱变动）

```
Decision → Position → Execution
```

## 主线3：反馈主线（防止亏死）

```
Execution → PnL → Feedback Loop → Risk → Decision
```

# 🔄 七、核心数据流（从数据到执行的完整链路）

```
原始数据
   ↓
[IngestionRuntime]  ← 数据采集 (Binance WS, 新闻, ETF)
   ↓
[Event Service]     ← 把"数据"变成"事件"（语义理解）
   ↓
[Fusion Service]    ← 多源信息融合（信息验证）
   ↓
[FeatureRuntime]    ← 特征计算
   ↓
[Factor Service]     ← 因子合成
   ↓
[RegimeRuntime]      ← 市场状态判断
   ↓
[Risk Engine]         ← 风险计算
   ↓
[Decision Service]   ← 交易决策
   ↓
[Position Service]   ← 仓位计算
   ↓
[ExecutionRuntime]   ← 订单执行
```

```
Execution   → 下单

Feedback    → 反思学习
Monitoring  → 看输赢
```
