# 🔄 二、各模块数据流详解

## 2.1 06_因子系统

```
输入采集来源：

┌─────────────────────────────────────────┐
│  1. 市场数据                              │
│  ├── BTC/ETH 价格（实时 + 历史）          │
│  ├── 成交量                               │
│  ├── 订单簿数据（可选）                   │
│  └── 来源：Binance / OKX API             │
├─────────────────────────────────────────┤
│  2. ETF数据                              │
│  ├── ETF资金流（inflow/outflow）         │
│  ├── 持仓量变化                           │
│  └── 来源：Farside / SoSoValue           │
├─────────────────────────────────────────┤
│  3. 宏观数据                             │
│  ├── 黄金价格                             │
│  ├── 原油价格                             │
│  ├── 美元指数                             │
│  └── 来源：Yahoo Finance / 第三方API     │
├─────────────────────────────────────────┤
│  4. 新闻/情绪数据                        │
│  ├── 新闻文本                             │
│  ├── 公告                               │
│  └── 来源：RSS / 新闻API / LLM解析       │
├─────────────────────────────────────────┤
│  5. 行为数据（可选）                     │
│  ├── 多空比                               │
│  ├── 爆仓数据                             │
│  └── 来源：交易所API / 第三方            │
└─────────────────────────────────────────┘
```

**处理流程：**

```python
原始数据 → 清洗 → 归一化 → 因子计算 → 标准化输出

因子类型：
├── trend_factor      # 趋势因子
├── flow_factor      # 资金流因子
├── sentiment_factor # 情绪因子（LLM）
├── macro_factor     # 宏观因子
├── behavioral_factor # 行为因子
└── historical_factor # 历史相似度因子
```

**输出格式：**

```json
{
  "factor_scores": {
    "trend": 0.55,
    "flow": -0.2,
    "sentiment": -0.6,
    "macro": 0.3,
    "behavioral": 0.4
  },
  "composite_score": 0.18,
  "regime": "RISK_OFF",
  "timestamp": 1710000000
}
```

## 2.3 08_仓位引擎设计

```
输入采集来源：

┌─────────────────────────────────────────┐
│  1. 决策信号（来自决策系统）             │
│  ├── signal: "LONG/SHORT/HOLD"         │
│  ├── confidence: 0.0-1.0               │
│  └── composite_score                    │
├─────────────────────────────────────────┤
│  2. 风险指标（来自风险模型）             │
│  ├── risk_index: 0-100                  │
│  ├── risk_level                        │
│  └── drivers                            │
├─────────────────────────────────────────┤
│  3. 当前持仓（来自State Engine）         │
│  ├── balance                            │
│  ├── positions[]                        │
│  └── unrealized_pnl                     │
├─────────────────────────────────────────┤
│  4. 市场波动（来自Feature Service）       │
│  └── atr: 0.025（2.5%）                │
└─────────────────────────────────────────┘
```

**处理流程：**

```python
signal + risk + 持仓 → base_size → risk_adjust → vol_adjust → leverage → SL/TP

仓位计算：
base_size = confidence × max_position
risk_factor = 1 - (risk_index / 100)
adjusted_size = base_size × risk_factor
vol_factor = target_vol / atr
final_size = adjusted_size × vol_factor
```

**输出格式：**

```json
{
  "symbol": "BTC",
  "action": "OPEN_LONG",
  "size": 0.18,
  "leverage": 2,
  "stop_loss": 66000,
  "take_profit": 72000,
  "confidence": 0.78,
  "risk_level": "medium"
}
```

## 2.4 09_决策系统

```
输入采集来源：

┌─────────────────────────────────────────┐
│  1. 因子数据（来自因子系统）             │
│  ├── factor_scores                      │
│  ├── composite_score                    │
│  └── regime                            │
├─────────────────────────────────────────┤
│  2. 风险数据（来自风险模型）             │
│  ├── risk_index                        │
│  └── max_position / max_leverage       │
├─────────────────────────────────────────┤
│  3. 市场状态（来自Regime Engine）         │
│  ├── current_regime                    │
│  ├── regime_confidence                 │
│  └── regime_rules                      │
├─────────────────────────────────────────┤
│  4. 组合风险（来自Portfolio Risk）       │
│  ├── total_exposure                    │
│  └── correlation_risk                   │
└─────────────────────────────────────────┘
```

**处理流程：**

```python
因子 + 风险 + Regime → 信号生成 → 信号质量过滤 → 最终决策

决策矩阵：
regime + signal + risk → ALLOW/HOLD/REJECT
```

**输出格式：**

```json
{
  "signal": "BUY",
  "confidence": 0.78,
  "direction": "LONG",
  "reason": "trend_aligned_with_flow",
  "quality_grade": "B+",
  "risk_adjusted": true
}
```

## 2.5 11_组合风险引擎

```
输入采集来源：

┌─────────────────────────────────────────┐
│  1. 持仓信息（来自State Engine）         │
│  ├── 各资产仓位                        │
│  ├── 杠杆                              │
│  └── 方向（LONG/SHORT）                │
├─────────────────────────────────────────┤
│  2. 资产风险（来自风险模型）             │
│  ├── 各资产波动率                      │
│  └── 各资产beta                       │
├─────────────────────────────────────────┤
│  3. 市场数据（来自Data Service）         │
│  ├── 价格序列                          │
│  └── 相关性矩阵                        │
├─────────────────────────────────────────┤
│  4. Regime状态                          │
│  └── 当前市场状态                       │
└─────────────────────────────────────────┘
```

**处理流程：**

```python
持仓 + 风险 + 相关性 → exposure → var/cvar → concentration → action

关键指标：
├── total_exposure（总暴露）
├── net_exposure（净暴露）
├── gross_exposure（总暴露）
├── correlation_risk（相关性风险）
├── var_95 / cvar_95（尾部风险）
└── hhi（集中度）
```

**输出格式：**

```json
{
  "portfolio_risk": {
    "total_exposure": 0.65,
    "correlation_risk": "HIGH",
    "var_95": 0.05,
    "es_95": 0.08,
    "action_required": "REDUCE",
    "action_amount": 0.15
  }
}
```

## 2.6 12_信号质量评分

```
输入采集来源：

┌─────────────────────────────────────────┐
│  1. 策略信号（来自决策系统）             │
│  ├── signal                            │
│  ├── direction                         │
│  └── confidence                        │
├─────────────────────────────────────────┤
│  2. 因子数据（来自因子系统）             │
│  ├── factor_scores                     │
│  └── composite_score                   │
├─────────────────────────────────────────┤
│  3. 历史表现（来自State/回测）           │
│  ├── 信号准确率                        │
│  ├── 盈亏比                            │
│  └── 胜率                              │
├─────────────────────────────────────────┤
│  4. 市场数据（来自Data Service）         │
│  ├── upcoming_events                  │
│  └── signal_age                        │
└─────────────────────────────────────────┘
```

**处理流程：**

```python
信号 + 因子 + 历史 → 各维度评分 → 加权 → 综合评分 → 执行建议

评分维度：
├── factor_alignment    (0.25)
├── signal_strength     (0.20)
├── regime_match        (0.20)
├── risk_adjusted       (0.15)
├── timing             (0.10)
└── fundamental_support (0.10)
```

**输出格式：**

```json
{
  "overall_score": 0.78,
  "grade": "B+",
  "is_executable": true,
  "execution_priority": "HIGH",
  "components": {
    "factor_alignment": 0.85,
    "signal_strength": 0.72,
    "regime_match": 1.00
  }
}
```
