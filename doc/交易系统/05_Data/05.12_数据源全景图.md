## 核心数据源详情

### 1. Binance WebSocket 🔴 S级

**地位**: 系统最核心数据源

**优势**:
- 免费
- 实时
- 稳定
- 低延迟

**数据流**:
```
Binance WebSocket
       ↓
  Kafka Topic
       ↓
  Feature Engine
       ↓
  Strategy/Risk
```

**支持数据类型**:
- [x] aggTrade - 聚合成交
- [x] bookTicker - 最优买卖价
- [x] liquidation - 强平
- [x] markPrice - 标记价格
- [x] fundingRate - 资金费率
- [ ] kline - K线
- [ ] depth - 订单簿

**配置文件**: `collectors/binance_websocket.py`

---

### 2. Telegram 🔴 S级

**地位**: 真正的 Alpha 来源

**优势**:
- 免费
- 稳定
- 低延迟
- bot 少
- 长文本

**监控频道**:
- WuBlockchain (币世界)
- 金十数据
- PANews
- Odaily
- TreeNews

**配置文件**: `collectors/telegram_adapter.py`

---

### 3. QQ 🟢 A级

**地位**: 中文加密社区核心来源

**优势**:
- 免费
- 中文社群活跃度高
- 有很多专业加密 QQ 群
- 信息扩散速度快
- 实时性好

**监控对象**:
- 币圈讨论群
- DeFi 研究群
- KOL 专属群
- 交易所官方群
- 链上分析群

**推荐框架**:
- [go-cqhttp](https://github.com/Mrs4s/go-cqhttp) (推荐)
- Mirai
- QQ 机器人框架

**配置文件**: `collectors/qq_adapter.py`

**环境变量**:
```bash
export QQ_HTTP_API_URL="http://127.0.0.1:5700"
```

---

### 4. RSS Feed 🟢 A级

**地位**: 稳定的低成本输入层

**支持源**:
- CoinDesk
- Blockworks
- Odaily RSS
- PANews RSS
- 金十 RSS

**配置文件**: `collectors/news_feed_collector.py`

---

### 5. Reddit 🟢 A级

**地位**: 被低估的深度来源

**优势**:
- 免费
- 稳定
- 长文本
- 情绪质量高
- bot 少

**监控子版**:
- r/CryptoCurrency
- r/Bitcoin
- r/ethtrader
- r/algotrading

**配置文件**: `collectors/social_media_collector.py`

---

### 6. Odaily Skill 🟡 B级

**地位**: 结构性中文新闻

**优势**:
- 无需 API Key
- 结构化数据
- M1-M5 模块覆盖

**模块**:
- M1: 今日必关注
- M2: 市场分析
- M3: 明日关注
- M4: 巨鲸追踪
- M5: API 原始数据

**配置文件**: `adapters/skill_adapter.py`

---

### 7. Twitter Push 🟡 B级

**地位**: 高优先级补充源

**优势**:
- 超低延迟
- 免费
- P0 账号监控

**限制**:
- 只能监控账号
- 通知数量有限
- 可能被 shadowban

**P0 账号**:
- @elonmusk
- @cz_binance
- @VitalikButerin
- @saylor
- @BarrySilbert
- @binance
- @okx
- @coinbase
- @EricBalchunas
- @WatcherGuru

**配置文件**: `twitter_push_server.py`

```
                    │  binance.market.*    │
                    └──────────┬───────────┘
                               │
        ┌─────────────────────┼─────────────────────┐

┌───────▼───────┐    ┌───────▼───────┐    ┌───────▼───────┐

        │                     │                     │
        └─────────────────────┼─────────────────────┘
                              ▼
                     ┌──────────────────┐
                     │  Event Normalize │
                     │  (StandardEvent) │
                     └────────┬─────────┘
                              │
              ┌───────────────┼───────────────┐
```

## 环境变量配置

```bash
# ============================================
# Telegram API (S级)
# ============================================
export TG_API_ID="your_api_id"
export TG_API_HASH="your_api_hash"

# ============================================
# Twitter API (B级)
# ============================================
export TWITTER_API_KEY="your_key"
export TWITTER_API_SECRET="your_secret"

# ============================================
# CryptoPanic API (A级)
# ============================================
export CRYPTOPANIC_API_KEY="your_key"

# ============================================
# Whale Alert API (B级)
```
