# Telegram 配置指南

## 快速开始

### 1. 获取 Telegram API

1. 访问 https://my.telegram.org/apps
2. 登录你的 Telegram 账号
3. 点击 "API development tools"
4. 填写应用信息：
   - App title: Trade Agent
   - Short name: trade_agent
   - Platform: Other
5. 点击 "Create application"
6. 记录你的 `api_id` 和 `api_hash`

### 2. 安装依赖

```bash
pip install telethon
```

### 3. 配置环境变量

```bash
export TELEGRAM_API_ID=12345678
export TELEGRAM_API_HASH=abcdef1234567890abcdef1234567890
export TELEGRAM_PHONE=+8613800138000
```

### 4. 测试连接

```bash
cd backend
python3 services/data_service/examples/telegram_mvp.py
```

## 关键词优先级

和 QQ 类似，Telegram 也使用 P0/P1/P2 优先级：

### P0 - 重要 (🔴)

```
SEC, ETF, approval, approved, rejected
爆仓, hack, 被盗, 跑路, 监管, 政策
崩盘, 暴跌, 暴涨, liquidation, FTX, Binance
```

### P1 - 关注 (🟡)

```
BTC, ETH, SOL, 上涨, 利好, 买入, 突破
bullish, breakout, pump, 做多, 牛市
```

### P2 - 普通 (🟢)

```
分析, 觉得, 可能, 讨论, 看看
```

## 代码示例

### 基本使用

```python
from services.data_service.sources.telegram_realtime import TelegramRealtimeSource

source = TelegramRealtimeSource()

async def on_event(event):
    print(f"[{event.metadata['priority']}] {event.channel}: {event.content}")

source.on_event = on_event

await source.listen()
```

### 带过滤

```python
source = TelegramRealtimeSource(channels=["WuBlockchain", "PANews"])

async def on_event(event):
    # 只处理 P0 和 P1
    if event.metadata['priority'] in ['P0', 'P1']:
        print(f"重要消息: {event.content}")

source.on_event = on_event
await source.listen()
```

### 发送 Telegram Bot 通知

```python
import os
import aiohttp

source = TelegramRealtimeSource()

TELEGRAM_BOT_TOKEN = os.getenv("TELEGRAM_BOT_TOKEN")
TELEGRAM_CHAT_ID = os.getenv("TELEGRAM_CHAT_ID")

async def on_event(event):
    if event.metadata['priority'] == 'P0':
        message = f"🔴 P0: {event.content[:200]}"
        await send_telegram(message)

async def send_telegram(message):
    url = f"https://api.telegram.org/bot{TELEGRAM_BOT_TOKEN}/sendMessage"
    data = {
        "chat_id": TELEGRAM_CHAT_ID,
        "text": message,
        "parse_mode": "HTML"
    }
    async with aiohttp.ClientSession() as session:
        await session.post(url, json=data)

await source.listen()
```

## 常见问题

### Q: 连接失败？

A: 检查：
1. API credentials 是否正确
2. 手机号是否带国际区号 (+86)
3. 是否在 https://my.telegram.org 开启了 2FA

### Q: 收不到消息？

A: 确认：
1. 你的账号已加入要监控的频道
2. 频道名或用户名正确
3. 频道是公开的或有权限

### Q: 如何找到频道用户名？

A:
1. 打开 Telegram
2. 进入频道
3. 点击频道名称
4. 滚动到底部，有 "链接" 一项
5. 公开频道的链接格式是 `t.me/username`

## 安全提示

1. **不要泄露 API credentials**
2. **使用小号**而非主号
3. **定期轮换** API keys
4. **遵守 Telegram ToS**，不要滥用
