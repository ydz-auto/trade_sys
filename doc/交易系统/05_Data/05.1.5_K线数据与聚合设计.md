# aggregation_service 负责：

## 聚合 feature

例如：

---

# Spread

Spread = Ask_{best} - Bid_{best}

---

# Mid Price

Mid = \frac{Ask_{best} + Bid_{best}}{2}

---

# Imbalance

Imbalance = \frac{BidVolume - AskVolume}{BidVolume + AskVolume}
