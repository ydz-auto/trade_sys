## 环境变量配置

### 快速配置

创建 `.env` 文件或直接在环境中设置：

```bash
# ============================================
# Twitter/X API
# ============================================
export TWITTER_API_KEY="your_api_key"
export TWITTER_API_SECRET="your_api_secret"
export TWITTER_ACCESS_TOKEN="your_access_token"
export TWITTER_ACCESS_TOKEN_SECRET="your_access_token_secret"

# ============================================
# CryptoPanic API
# ============================================
export CRYPTOPANIC_API_KEY="your_api_key"

# ============================================
# Whale Alert API
# ============================================
export WHALE_ALERT_API_KEY="your_api_key"

# ============================================
# 其他可选配置
# ============================================
export TWITTER_MONITOR_ACCOUNTS="elonmusk,cz_binance,VitalikButerin"
export DATA_FETCH_INTERVAL=60  # 秒
```

## 各数据源配置详解

### 1. ClawHub Odaily Skill (无需 API Key)

**状态**: ✅ 已接入，无需额外配置

**说明**: 
- 自动从 ClawHub 获取加密市场新闻
- 包含 5 个模块：M1-M5
- 安装方式：`openclaw skills install odaily-skill`

**文档**: https://clawhub.ai/odaily/odaily-skill

---

### 2. Twitter/X API

**状态**: ✅ 已接入，需 API Key

#### 获取 API Key

1. 访问 [Twitter Developer Portal](https://developer.twitter.com/en/docs/twitter-api)
2. 注册开发者账号并创建应用
3. 获取以下凭证：
   - API Key (Consumer Key)
   - API Secret (Consumer Secret)
   - Access Token
   - Access Token Secret

#### 配置步骤

```bash
# 1. 获取凭证后设置环境变量
export TWITTER_API_KEY="xZ6...your_key"
export TWITTER_API_SECRET="y8Q...your_secret"
export TWITTER_ACCESS_TOKEN="123...your_token"
export TWITTER_ACCESS_TOKEN_SECRET="abc...your_token_secret"
```

#### 功能说明

- 支持监控 9 个重要账号：
  - @elonmusk
  - @cz_binance
  - @VitalikButerin
  - @saylor
  - @BarrySilbert
  - @binance
  - @okx
  - @coinbase
  - @SBF_FTX
- 自动提取币种 hashtags
- 基于互动量计算重要性

---

### 3. CryptoPanic API

**状态**: ✅ 已接入，需 API Key

#### 获取 API Key

1. 访问 [CryptoPanic Developers](https://cryptopanic.com/developers/api/)
2. 注册并获取免费 API Key

#### 配置步骤

```bash
# 设置 API Key
export CRYPTOPANIC_API_KEY="your_api_key"
```

#### 功能说明

- 免费限额：100 请求/小时
- 支持情绪过滤
- 包含多种来源新闻

---

### 4. Whale Alert API

**状态**: ✅ 已接入，需 API Key

#### 获取 API Key

1. 访问 [Whale Alert API Docs](https://docs.whale-alert.io/)
2. 注册并获取 API Key

#### 配置步骤

```bash
# 设置 API Key
export WHALE_ALERT_API_KEY="your_api_key"
```

#### 功能说明

- 免费限额：10 请求/分钟
- 监控大额链上转账
- 支持 BTC, ETH 等主流链
- 最小监控金额：100,000 USD

---

### 5. PANews (可选)

**状态**: ⚠️ 可扩展

需要时可接入 PANews Skill，类似 Odaily 的方式。

## 常见问题

### Q: 没有 API Key 可以运行吗？

A: 可以，系统会自动使用模拟数据。

### Q: Twitter API 免费额度够吗？

A: 免费套餐通常足够开发使用，生产环境建议升级套餐。

### Q: 如何确认 API Key 是否生效？

A: 运行 `test_multi_source.py` 查看是否显示 "(使用真实 API)"。
