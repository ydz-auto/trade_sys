## 🌍 公网访问方案

### 方案一: 使用 ngrok（推荐，简单快捷）

1. **安装 ngrok**
   ```bash
   brew install ngrok  # Mac
   # 或去 https://ngrok.com 下载
   ```

2. **启动 ngrok 隧道**
   ```bash
   # 转发前端
   ngrok http 3002

   # 转发 API (可选)
   ngrok http 8001
   ```

3. **访问**
   ngrok 会给你一个公网地址，例如: `https://xxxx-xx-xx-xx-xx.ngrok-free.app`

---

### 方案二: 使用 frp（自托管，稳定）

1. **需要一台公网服务器**（阿里云/腾讯云等）

2. **服务端配置 (frps)**
   ```ini
   [common]
   bind_port = 7000
   vhost_http_port = 80
   vhost_https_port = 443
   ```

3. **客户端配置 (frpc)**
   ```ini
   [common]
   server_addr = 你的服务器IP
   server_port = 7000

   [web]
   type = http
   local_port = 3002
   custom_domains = 你的域名
   ```

4. **访问**: 绑定的域名

---

### 方案三: 使用 Tailscale (VPN)

1. **安装 Tailscale**
   ```bash
   curl -fsSL https://tailscale.com/install.sh | sh
   ```

2. **启动**
   ```bash
   sudo tailscale up
   ```

3. **获取 Tailscale IP**，通过这个 IP 从任何地方访问

---

### 方案四: 云服务器部署（最稳定）

1. **买一台云服务器** (阿里云/腾讯云/AWS等)
2. **安装 Docker 和 Docker Compose**
3. **部署完整服务**

优势: 24小时在线，稳定可靠
